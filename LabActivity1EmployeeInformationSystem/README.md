Employee Information SystemThis Java program is a basic Employee Information System that simulates collecting personal and work-related input from the user, as well as calculating the employee's daily salary. This program was created for a laboratory activity.FeaturesUser Input: Gathers employee information through interactive console prompts.Daily Salary Calculation: Computes daily salary based on user-provided hours and hourly wage.Formatted Output: Displays employee details and daily salary, with the salary formatted to two decimal places.How to Compile and RunTo compile and run this Java application, follow these steps:Save the file: Save the code as LabActivity1EmployeeInformationSystem.java.Open a terminal or command prompt.Navigate to the directory where you saved the file.Compile the Java code:javac LabActivity1EmployeeInformationSystem.java
Run the compiled program:java LabActivity1EmployeeInformationSystem
Example UsageWhen you run the program, it will prompt you to enter the following information:Enter your first name: [e.g., Charles]
Enter your last name: [e.g., Garcia]
Enter your age: [e.g., 18]
Enter hours worked: [e.g., 8.5]
Enter hourly wage: [e.g., 75.00]
After you provide all the inputs, the program will output the employee's information and daily salary:Employee Information
---------------------

Age         : 18 years old
